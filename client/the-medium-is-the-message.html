Meteor keeps it simple

You're 4 commands away from your first Meteor app. 


  curl https://install.meteor.com | /bin/sh
	
  meteor create the-medium-is-the-message
	
  cd the-medium-is-the-message

  meteor

  • Uniform data access.
    1 api on client & server means less to learn.
  • Data over the wire.
    Push data; don't wait for a reloads.

Transparent syncing:

  • Reactive views. UI updates as data is received.
  • Latency compensation.
    Local changes, instant updates:
    Mongo syncs and patches in the background.
